# Data Sources
## 1. U.S. Census Bureau - Foreign-Born Population
- URL: https://www.census.gov/topics/population/foreign-born.html
- Use: Labor force demographics, population trends

## 2. U.S. Patent and Trademark Office (USPTO)
- URL: https://www.uspto.gov/ip-policy/economic-research/research-datasets
- Use: Patent filings by immigrant inventors, innovation metrics

## 3. Congressional Budget Office (CBO)
- URL: https://www.cbo.gov/topics/immigration
- Use: GDP contribution projections, fiscal impact analysis

## 4. Kauffman Foundation - Entrepreneurship Data
- URL: https://www.kauffman.org/entrepreneurship/research/
- Use: Immigrant-founded businesses, startup rates, job creation

## 5. DHS Immigration Statistics Yearbook (Historical)
- URL: https://www.dhs.gov/immigration-statistics/yearbook
- Use: Historical immigration trends (1820-present), Industrial Revolution context
