# The Positive-Sum Economic Engine: How Immigration Grows the U.S. Economy

## Project Goal
A Power BI Dashboard demonstrating that immigration is a net economic gain for the U.S., creating wealth and opportunities through the 'positive-sum' model.

## Key Thesis
The economic pie GROWS through immigration — it is not fixed or zero-sum.

## Structure
- `data/` — Raw and processed datasets
- `docs/` — Data sources, visualizations, narrative strategy
- `visuals/` — Power BI files and exports
